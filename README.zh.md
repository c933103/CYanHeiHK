昭源甄黑體 (CYanHeiHK)
====================

## 關於

### 「昭源甄黑體」是……

「昭源甄黑體」是一款從「思源黑體」修改而成、適用於香港地區的中文字體。

「思源黑體」由 Adobe 和 Google 合作開發（Google 以 Noto Sans CJK 名義發佈），提供多種語系版本，當中繁體中文版本的字形，是遵照台灣教育部的「[國字標準字體](https://zh.wikipedia.org/zh-hk/%E5%9C%8B%E5%AD%97%E6%A8%99%E6%BA%96%E5%AD%97%E9%AB%94)」字形標準製作，不符合香港字體標準。

### 關於「香港字體標準」

目前香港有兩套與字形相關的標準（或指引）：
1. 由教育局公佈的《常用字字形表》，最新版於 2007 年以《香港小學學習字詞表》附錄形式發表。
2. 由政府資訊科技總監辦公室公佈的《中文電腦字形參考指引》，在 [2002 年 2 月](http://www.info.gov.hk/gia/general/200202/27/0227167.htm) 發表。

二者分別如下：
1. 《常用字字形表》規範手寫楷書字形（不包括電腦字形），提供四千多字的標準字形。
2. 《中文電腦字形參考指引》規範電腦字體字形，並提供宋、楷兩種字體的標準字形文件。做法是將《常用字字形表》(2000年修訂本) 中的標準書體為基礎，將當中各字的部件進行拆分、整理、歸類，再將整理出來的原則，擴展至其他漢字。例如雖然《常用字字形表》不收「嘅」字，但此字由「口」部「既」字組成，而《字形表》有這兩個部件的標準寫法（如「既」不作「旣」），《指引》便可據此制定「嘅」的標準字形。《指引》涵蓋 ISO 10646 中，基本多重平面 (BMP)、Ａ擴充集 (Ext-A) 和香港增補字符集(HKSCS) 2001 版的漢字（後來再以附錄形式補上 HKSCS-2005 的資料）。
3. 雖然《指引》是依據《字形表》製作，但由於二者是由不同部門異步進行，更新時間不同，加上《指引》中某些部件的寫法也有調整，所以少數字元的部件寫法並不一致。

不過，香港用於印刷品的商用字體，幾乎沒有一種是完全依據香港字形標準設計。這有幾個可能原因：
1. 多數電腦字體在八十年代生產，早於《指引》十多年；而調整、修改字形以符合《指引》標準未必合符成本原則。
2. 《指引》以楷書為主，追求同一部件在其他書體（如宋體）寫法必須保持一致，不會考慮原屬楷書的筆法套用到其他書體是否美觀。
3. 一些不符《指引》的寫法，大眾已經習慣；商業字體在調整寫法前不能不多考慮大眾認受性和使用標準寫法對字體美觀性的影響。

所以，原來據台灣字形標準製作的「思源黑體」，對香港用者來說，有兩個問題：
1. 一些字元的字形未必符合香港書寫標準；
2. 有時候，一些字元符合香港書寫標準，卻未必符合香港人的習慣。

### 目標

* 「思源黑體」將會製作符合即將公佈的《香港電腦漢字參考字形》更新版的香港繁體中文字體。
* 「昭源甄黑體」可作為符合香港字形標準的「思源黑體」推出前的暫代方案。而「思源黑體」更新後，「昭源甄黑體」仍有其價值，因其試圖平衡香港的「標準書體」和「日常習慣」，取態跟商用字體接近，或能滿足一些人的需要。

## 與原版思源黑體的分別

「昭源甄黑體」修改自「思源黑體（台灣繁體）」1.004版。「修改」包括「修改字碼對映」或「修改字形」兩種情況。原版「思源黑體」提供適用於日文、韓文、簡體、繁體的四種字體風格。如果某一字元的台灣版本不適用於香港，而其他地區（如日本）版本適用時，便可透過修改字碼對映 (Remap) 解決。如果沒有地區的字形適合，就要修改字形。

修改後的「昭源甄黑體」與原版中文字形區別如下：

1. 一些字形改為符合香港標準；
2. 一些字形改為或保留較常見的寫法；
3. 一些部首改為較合於黑體使用的式樣；
4. 「辶」部件重新設計；

另外也有跟中文字形無關的調整：

1. 個別全形標點（，。、：；！？）略為收細；
2. 修改半形 1 和 g 的外觀。

## 修訂過程

礙於人力物力，逐一檢查原字體所有字形（超過四萬）再修改並不可行，也不切實際。「思源黑體」台灣版中，符合台灣繁體寫法的字其實只涵蓋 Big5 和 HKSCS 字集，但這個範圍 (超過 17,600 字元) 仍然太大。經審視後，決定將檢查範圍收窄如下： 

1. Big5 + HKSCS 的 17,462 字是最大支援範圍。不在這兩個編碼集的字元不會處理。
2. 教育局[香港小學學習字詞表網頁](http://www.edbchinese.hk/lexlist_ch/index.htm) 網頁收載的 4,805 字元會主動審視並修改。
3. IICORE 收錄，有香港標籤的 5,224 個中文字（[來源](http://www.unicode.org/L2/L2010/10375-02n4153-files/IICORE.txt)）亦會主動審視並修改，當中有 456 個中文字不見於上表。
4. IICORE 收錄，有台灣、澳門、日本標記的漢字會審視，是否修改視乎有無可用的對映。
5. 其他漢字會審視但未必會修改。

為節省時間，修改時盡量會選擇「修改字碼對映」（尤其對非常用或不在香港 IICORE 的字），當中或會忽略一些細微但「思源黑體」原版視為地區分別的字形特徵，例如：

![Region level variants](doc/images/region-level-variants.png?raw=true "Region-level variants ignored in this font")

修改後的字體相信足敷香港使用。留意本字體的字數跟原版「思源黑體」language specific 版本相同（逾四萬字），只是當中未修改的字形，**可能**不符香港使用習慣，例如「縎」字。

## 下載

請前往[發佈](https://github.com/tamcy/CYanHeiHK/releases)頁下載，有桌面字型（OTF格式，兩個版本）和一套網頁字型（TTF/WOFF/WOFF2三種格式）可供選擇。

### 桌面字型

本字體提供兩種桌面字體版本。由於字體名稱相同，不能同時安裝兩種版本。
  
* `CYanHeiHK_{版本}.7z` 是「正常版本」的 OpenType (.OTF) 字體，建議一般使用。
* `CYanHeiHK_{版本}_adjusted_fontbbox.7z` 是調整 “Font BBox” 參數後的版本。本字體承繼思源黑體，擁有較大的 Font BBox 值，個別應用程式可能會出現問題。請在確實明白這個版本的用意時才考慮使用，否則請選用上述「正常版本」。
 
### 網頁字型

`CYanHeiHK_{版本}_subset.7z` 是從桌面字型版本抽取常用者（約 8,010 字元），以 WOFF/WOFF2 發佈，適合網頁使用；並提供收字量相同的 TTF 版本作其他用途（如附在流動應用程式但又不想用較大的 OTF 版本）。

中文字元是根據以下條件選出： 
   * IICORE 中有香港/澳門/台灣/日本來源標記的字會預設收入，但罕用者會人手篩走。日本源只收有 Big5/HKSCS 編碼的字。
   * 一些香港常用而缺 IICORE 標記的字（主要是 HKSCS 字，如㗅、喆）人手收入。

字型亦包括以下非中文字元：
   * ASCII 文字（TTF 中之 “no-latin” 版本除外）。  
   * 標點、日文平假、片假名。
